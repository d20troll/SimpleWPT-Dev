# Screenshot

For the theme screenshot (screenshot.png), create an image with these specifications:

- **Size**: 1200x900 pixels
- **Format**: PNG
- **Content**: A clean mockup showing the theme's design
- **Suggested elements**:
  - Header with logo/site title
  - Navigation menu
  - Featured blog post with image
  - Clean typography showcase
  - Footer area

You can use a design tool like Figma, Canva, or Photoshop to create this.

**Color palette to use**:
- Primary: #2563eb
- Secondary: #7c3aed
- Accent: #f59e0b
- Dark: #1f2937
- Light Gray: #f3f4f6
- White: #ffffff

The screenshot should represent the clean, modern aesthetic of the theme.
